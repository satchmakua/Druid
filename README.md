# Druid

**An immutable, provable record of what the government's environmental data said — and how it is being changed.**

Druid continuously observes a curated set of high-value U.S. government environmental
pages and datasets, cryptographically attests every observation in a tamper-evident
transparency log, detects *meaningful* change (definition swaps, threshold edits, silent
dataset shifts — not byte noise), and emits a public, citable, self-verifying record.
Its differentiator: a **downloadable proof bundle** anyone can verify **offline**,
trusting neither the government nor Druid's operators.

Complementary to the volunteer rescue ecosystem (EDGI, PEDP, End-of-Term, Data Rescue
Project) — adding the two things none of them treat as primary: **provable observation
integrity** and **classified manipulation detection**.
